RIS Tutorials at MSI

This site will list information about the tutorials that are developed and hosted by RIS at the University of Minnesota Supercomputing Institute. Note that this site will not support registration, nor will it give material for tutorials at MSI generally. This page is internally maintained for easy access and updating of the RIS tutorials.

Active Tutorials

These tutorials are actively maintained and delivered on a regular basis.

RNAseq (Workshop)
This tutorial is a hands-on tutorial for analysis of bulk RNAseq data for differential gene expression. It uses command line tools on the MSI compute cluster.

RNAseq (Lecture)
This tutorial is a lecture on the principles of bulk RNAseq analysis, including considerations for molecular biology protocols, sequencing technologies, experimental design, and an overview of the University of Minnesota Genomics Center.

Stratus and dbGaP
This tutorial is a hands-on tutorial for setting up a "Stratus" environment on MSI, which is our service for computing with protected data. Note that you must purchase a Stratus subscription to use this service, and you must have an active dbGaP project to be eligible.
